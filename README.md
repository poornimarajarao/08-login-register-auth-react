# React Authentication Controlled Forms
## What I Learned

* React app with controlled forms and conditional rendering.
* Managing form inputs using the `useState` hook and controlled components.
* Implementing basic form validation such as username length checks and password matching
* Dynamically showing or hiding components based on app state.
* Handling form submissions and preventing default browser behavior.
* Passing functions as props to control UI (e.g switching between Login and Register forms).
* Creating responsive UI layouts using Bootstrap classes.
